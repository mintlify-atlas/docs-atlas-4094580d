# Convolutional Neural Network Models

All ConvNeXt models are trained with:
- **Extended training**: 600 epochs
- **AMP**: Automatic Mixed Precision for efficiency
- **EMA or non-EMA checkpoints**: Selected for best performance
- **Modern training recipes**: Following TorchVision's latest best practices

## Common Usage Patterns

### Fine-tuning

```python
import torch

# Load pretrained model
model = torch.hub.load('B-cos/B-cos-v2', 'resnet50', pretrained=True)

# Get feature extractor and classifier
feature_extractor = model.get_feature_extractor()
classifier = model.get_classifier()

# Freeze feature extractor
for param in feature_extractor.parameters():
    param.requires_grad = False

# Fine-tune only the classifier
for param in classifier.parameters():
    param.requires_grad = True
```

### Generating Explanations

```python
import torch
from PIL import Image

model = torch.hub.load('B-cos/B-cos-v2', 'densenet121', pretrained=True)
model.eval()

img = model.transform(Image.open('image.jpg'))
img = img[None].requires_grad_()

# Get explanation
expl_out = model.explain(img)
print(f"Prediction: {expl_out['prediction']}")
print(f"Explanation shape: {expl_out['explanation'].shape}")
```
