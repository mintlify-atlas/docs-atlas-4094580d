## Prerequisites

Before you begin, ensure you have:
- Python 3.7 or higher
- PyTorch 1.13 or higher
- torchvision

## Installation

Install the B-cos package using pip:

<CodeGroup>
```bash pip
pip install bcos
```

```bash conda
conda install -c conda-forge bcos
```
</CodeGroup>

<Note>
  For training models from scratch, see the [Installation Guide](/installation) for the full development environment setup.
</Note>

## Load a pretrained model

Load any of the 40+ pretrained B-cos models via `torch.hub`:

```python
import torch

# List all available models
available_models = torch.hub.list('B-cos/B-cos-v2')
print(f"Available models: {len(available_models)}")

# Load a pretrained B-cos ResNet-50
model = torch.hub.load('B-cos/B-cos-v2', 'resnet50', pretrained=True)
model.eval()
```

<Accordion title="See all available models">
- **ResNets**: resnet18, resnet34, resnet50, resnet101, resnet152, resnext50_32x4d
- **DenseNets**: densenet121, densenet161, densenet169, densenet201
- **ConvNeXt**: convnext_tiny, convnext_base, convnext_tiny_bnu, convnext_base_bnu
- **Vision Transformers**: simple_vit_ti/s/b/l_patch16_224, vitc_ti/s/b/l_patch1_14
- **VGG**: vgg11_bnu

See the [Model Zoo](/model-zoo/overview) for complete details.
</Accordion>

## Generate explanations

Use the built-in `.explain()` method to visualize model predictions:

```python
from PIL import Image
import matplotlib.pyplot as plt

# Load and preprocess an image
img = Image.open('cat.jpg')
img_tensor = model.transform(img)  # Built-in preprocessing
img_tensor = img_tensor[None].requires_grad_()

# Generate prediction and explanation
expl_out = model.explain(img_tensor)

# Display results
print(f"Predicted class: {expl_out['prediction']}")
print(f"Confidence: {expl_out['logit_output'].max():.2f}")

# Visualize explanation
plt.figure(figsize=(10, 5))
plt.subplot(1, 2, 1)
plt.imshow(img)
plt.title('Input Image')
plt.axis('off')

plt.subplot(1, 2, 2)
plt.imshow(expl_out['explanation'])
plt.title('B-cos Explanation')
plt.axis('off')
plt.show()
```

<Info>
The explanation heatmap shows which regions of the input contributed most to the model's prediction. Red/warm colors indicate positive contributions, while blue/cool colors indicate negative contributions.
</Info>

## Understanding the output

The `.explain()` method returns a dictionary with:

| Key | Type | Description |
|-----|------|-------------|
| `prediction` | int | Predicted class index |
| `logit_output` | Tensor | Raw logit scores for all classes |
| `explanation` | ndarray | RGB+alpha explanation image (H, W, 4) |
| `contribution_map` | Tensor | Spatial contribution map (1, H, W) |
