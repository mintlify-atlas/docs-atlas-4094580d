## Overview

B-cos provides utilities to visualize the explanations generated by its models. These functions convert gradient information into interpretable images and plots.

## gradient_to_image

Converts the dynamic linear mapping (gradients) from a B-cos model into a color RGBA image that visually represents the explanation.

### Signature

```python
def gradient_to_image(
    image: Tensor,
    linear_mapping: Tensor,
    smooth: int = 15,
    alpha_percentile: float = 99.5
) -> np.ndarray
```

### Parameters

<ParamField path="image" type="torch.Tensor" required>
  Original input image encoded with 6 color channels (RGB and inverse RGB). Shape: `[C, H, W]` where `C=6`.
</ParamField>

<ParamField path="linear_mapping" type="torch.Tensor" required>
  Linear mapping W (from input to logit layer) of the B-cos model (typically `input.grad`). Shape: `[C, H, W]`, same as image.
</ParamField>

<ParamField path="smooth" type="int" default="15">
  Kernel size for smoothing the alpha (transparency) values. Larger values produce smoother visualizations.
</ParamField>

<ParamField path="alpha_percentile" type="float" default="99.5">
  Cut-off percentile for the alpha value, in range [0, 100]. Controls the transparency scaling of the explanation.
</ParamField>

### Returns

<ResponseField name="explanation" type="np.ndarray">
  RGBA image explanation of the B-cos model. Shape: `[H, W, 4]` where the last dimension represents RGBA channels.
</ResponseField>

### Usage Example

```python
import torch
import bcos
import matplotlib.pyplot as plt

# Load model and prepare input
model = bcos.models.resnet18(pretrained=True)
model.eval()

# 6-channel input (RGB + inverse RGB)
input_tensor = torch.randn(1, 6, 224, 224, requires_grad=True)

# Generate explanation
with bcos.explanation_mode(model):
    output = model(input_tensor)
    pred_class = output.max(1).indices
    output[0, pred_class].backward()

# Convert gradients to visual explanation
explanation_img = bcos.gradient_to_image(
    image=input_tensor[0],
    linear_mapping=input_tensor.grad[0],
    smooth=15,
    alpha_percentile=99.5
)

# Display the explanation
plt.imshow(explanation_img)
plt.axis('off')
plt.title('B-cos Explanation')
plt.show()
```

### How It Works

1. **Contribution Calculation**: Computes the contribution map by multiplying the image with its gradients: `(image * linear_mapping).sum(0)`

2. **Direction Normalization**: Normalizes the gradient vectors to maintain direction while ensuring the max entry is 1

3. **RGB Computation**: Processes the 6-channel gradients (RGB and inverse) to produce final RGB values that sum to 1

4. **Alpha Channel**: Sets transparency based on the L2 norm of gradients, showing only positive contributions and applying smoothing

5. **Output**: Returns an RGBA image where color indicates the explanation direction and transparency indicates importance

## plot_contribution_map

Visualizes a contribution map as a heatmap, showing which spatial locations contribute positively or negatively to the model's prediction.

### Signature

```python
def plot_contribution_map(
    contribution_map: Union[Tensor, np.ndarray],
    ax: Optional[plt.Axes] = None,
    vrange: Optional[float] = None,
    vmin: Optional[float] = None,
    vmax: Optional[float] = None,
    hide_ticks: bool = True,
    cmap: str = "bwr",
    percentile: float = 99.5
) -> Tuple[plt.Axes, matplotlib.image.AxesImage]
```

### Parameters

<ParamField path="contribution_map" type="torch.Tensor | np.ndarray" required>
  2D matrix of contributions to visualize. Shape: `(H, W)` representing spatial locations.
</ParamField>

<ParamField path="ax" type="plt.Axes" default="None">
  Matplotlib axis on which to plot. If None, a new figure and axis are created.
</ParamField>

<ParamField path="vrange" type="float" default="None">
  If None, the colormap ranges from -v to v, where v is the maximum absolute value. If provided, the range is [-vrange, vrange], unless overridden by vmin or vmax.
</ParamField>

<ParamField path="vmin" type="float" default="None">
  Manually overwrite the minimum value for the colormap range instead of using -vrange.
</ParamField>

<ParamField path="vmax" type="float" default="None">
  Manually overwrite the maximum value for the colormap range instead of using vrange.
</ParamField>

<ParamField path="hide_ticks" type="bool" default="True">
  If True, hides axis ticks for a cleaner visualization.
</ParamField>

<ParamField path="cmap" type="str" default="bwr">
  Matplotlib colormap name to use. Default "bwr" (blue-white-red) is ideal for showing positive and negative contributions.
</ParamField>

<ParamField path="percentile" type="float" default="99.5">
  Percentile cut-off for clipping extreme values in the contribution map. Values beyond this percentile are clipped.
</ParamField>

### Returns

<ResponseField name="ax" type="plt.Axes">
  The matplotlib axis on which the contribution map was plotted.
</ResponseField>

<ResponseField name="im" type="matplotlib.image.AxesImage">
  The image object of the contribution map plot.
</ResponseField>

### Usage Example

```python
import torch
import bcos
import matplotlib.pyplot as plt

# Load model
model = bcos.models.resnet18(pretrained=True)
model.eval()

input_tensor = torch.randn(1, 6, 224, 224, requires_grad=True)

# Generate contribution map
with bcos.explanation_mode(model):
    output = model(input_tensor)
    pred_class = output.max(1).indices
    output[0, pred_class].backward()

# Calculate contribution map
contribution_map = (input_tensor * input_tensor.grad).sum(1)[0]  # Shape: [H, W]

# Visualize
ax, im = bcos.plot_contribution_map(
    contribution_map,
    cmap='bwr',
    percentile=99.5
)
plt.colorbar(im, ax=ax)
plt.title('Contribution Map')
plt.show()
```

### Advanced Usage: Multiple Subplots

```python
import matplotlib.pyplot as plt

fig, axes = plt.subplots(1, 3, figsize=(15, 5))

# Plot contribution maps for different classes
for idx, ax in enumerate(axes):
    with bcos.explanation_mode(model):
        output = model(input_tensor)
        output[0, idx].backward(retain_graph=True)
    
    contribution_map = (input_tensor * input_tensor.grad).sum(1)[0]
    
    ax_ret, im = bcos.plot_contribution_map(
        contribution_map,
        ax=ax,
        cmap='bwr'
    )
    ax.set_title(f'Class {idx}')
    
    # Clear gradients for next iteration
    input_tensor.grad.zero_()

plt.tight_layout()
plt.show()
```

### Customizing the Colormap

```python
# Use a different colormap and custom range
ax, im = bcos.plot_contribution_map(
    contribution_map,
    cmap='seismic',  # Alternative colormap
    vmin=-1.0,       # Fixed minimum
    vmax=1.0,        # Fixed maximum
    percentile=95.0  # Less aggressive clipping
)
```
