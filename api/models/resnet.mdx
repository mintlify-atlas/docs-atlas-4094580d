# ResNet Models

B-cos ResNet models are residual networks adapted for inherently interpretable predictions. These models provide competitive accuracy while maintaining full interpretability through B-cos alignment.

## Available Models

| Model | Acc@1 | Acc@5 | Params | Entrypoint |
|-------|-------|-------|--------|------------|
| ResNet-18 | 68.736% | 87.430% | 11.69M | `resnet18` |
| ResNet-34 | 72.284% | 90.052% | 21.80M | `resnet34` |
| ResNet-50 | 75.882% | 92.064% | 25.54M | `resnet50` |
| ResNet-101 | 76.532% | 92.538% | 44.50M | `resnet101` |
| ResNet-152 | 76.484% | 92.398% | 60.13M | `resnet152` |
| ResNeXt-50 32x4d | 75.820% | 91.810% | 25.00M | `resnext50_32x4d` |
| ResNet-50 (long) | 79.468% | 94.452% | 25.54M | `resnet50_long` |
| ResNet-152 (long) | 80.144% | 94.116% | 60.13M | `resnet152_long` |

## Usage

### Basic Usage

```python
import torch

# Load pretrained ResNet-50
model = torch.hub.load('B-cos/B-cos-v2', 'resnet50', pretrained=True)

# Load without pretrained weights
model = torch.hub.load('B-cos/B-cos-v2', 'resnet50', pretrained=False)
```

### Inference Example

```python
from PIL import Image

# Load and preprocess image
img = Image.open('image.jpg')
img_tensor = model.transform(img).unsqueeze(0)

# Get prediction
model.eval()
with torch.no_grad():
    output = model(img_tensor)
    pred = output.argmax(dim=1)
```
