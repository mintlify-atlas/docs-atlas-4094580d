# DenseNet Models

B-cos DenseNet models are densely connected convolutional networks adapted for inherently interpretable predictions. These models are parameter-efficient while maintaining competitive accuracy.

## Available Models

| Model | Acc@1 | Acc@5 | Params | Entrypoint |
|-------|-------|-------|--------|------------|
| DenseNet-121 | 73.612% | 91.106% | 7.95M | `densenet121` |
| DenseNet-161 | 76.622% | 92.554% | 28.58M | `densenet161` |
| DenseNet-169 | 75.186% | 91.786% | 14.08M | `densenet169` |
| DenseNet-201 | 75.480% | 91.992% | 19.91M | `densenet201` |
| DenseNet-121 (long) | 77.302% | 93.234% | 7.95M | `densenet121_long` |

## Usage

### Basic Usage

```python
import torch

# Load pretrained DenseNet-121
model = torch.hub.load('B-cos/B-cos-v2', 'densenet121', pretrained=True)

# Load without pretrained weights
model = torch.hub.load('B-cos/B-cos-v2', 'densenet121', pretrained=False)
```

### Inference Example

```python
from PIL import Image

# Load and preprocess image
img = Image.open('image.jpg')
img_tensor = model.transform(img).unsqueeze(0)

# Get prediction with explanation
model.eval()
img_tensor.requires_grad_()
expl_out = model.explain(img_tensor)
print(f"Predicted class: {expl_out['prediction']}")
```

## Notes

DenseNet models are particularly parameter-efficient. For example, DenseNet-121 achieves 73.6% top-1 accuracy with only 7.95M parameters, making it an excellent choice for resource-constrained environments.

The long-trained variant (`densenet121_long`) achieves significantly better performance (77.3% top-1) with the same architecture by using an extended training schedule (600 epochs) with TorchVision's improved training recipe.
