# Vision Transformer (ViT) Models

B-cos Vision Transformers adapt the ViT architecture for inherently interpretable predictions. Both B-cos and standard (non-B-cos) variants are available for comparison.

## Requirements

**Important:** ViT models require the `einops` package:

```bash
pip install einops
```

## Usage

### Basic Usage

```python
import torch

# Load B-cos ViT model
model = torch.hub.load('B-cos/B-cos-v2', 'simple_vit_b_patch16_224', pretrained=True)

# Load B-cos ViT with conv stem
model = torch.hub.load('B-cos/B-cos-v2', 'vitc_b_patch1_14', pretrained=True)

# Load standard (non-B-cos) ViT for comparison
model = torch.hub.load('B-cos/B-cos-v2', 'standard_simple_vit_b_patch16_224', pretrained=True)
```

### Inference Example

```python
from PIL import Image
import matplotlib.pyplot as plt

# Load and preprocess image
img = Image.open('image.jpg')
img_tensor = model.transform(img).unsqueeze(0).requires_grad_()

# Get prediction with explanation (B-cos models only)
model.eval()
expl_out = model.explain(img_tensor)
print(f"Predicted class: {expl_out['prediction']}")
plt.imshow(expl_out['explanation'])
plt.show()
```

## Notes

### Model Variants

**Simple ViT:** Standard Vision Transformer with 16x16 patch embeddings on 224x224 images.

**ViTC (ViT with Convolutional Stem):** Uses early convolutions before the transformer, which has been shown to help transformers see better. After the convolutional stem, the ViT processes 1x1 patches on a 14x14 feature map.

### B-cos vs Standard Models

B-cos ViT models are inherently interpretable and can generate explanations via the `.explain()` method. Standard (non-B-cos) models are provided for comparison and achieve slightly higher accuracy but lack built-in interpretability.

### Performance Notes

ViT models with convolutional stems (ViTC) generally achieve better accuracy than their simple counterparts, especially for smaller models. The B-cos ViTC-L achieves 77.78% top-1 accuracy, making it one of the highest-performing B-cos models.
