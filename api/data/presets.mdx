---
title: Transform Presets
description: Pre-configured transformation pipelines for ImageNet and CIFAR-10 datasets
---

Transform presets provide ready-to-use transformation pipelines optimized for common datasets. These presets handle all the necessary preprocessing steps including resizing, augmentation, and B-cos specific transformations.

## ImageNet Presets

### ImageNetClassificationPresetTrain

<ParamField path="ImageNetClassificationPresetTrain" type="class">
  Training transform preset for ImageNet classification with comprehensive augmentation options.

  <Expandable title="Parameters">
    <ParamField path="crop_size" type="int" required>
      Size for random cropping (typically 224 or 176).
    </ParamField>

    <ParamField path="mean" type="tuple" default="(0.485, 0.456, 0.406)">
      Channel means for normalization (ImageNet defaults).
    </ParamField>

    <ParamField path="std" type="tuple" default="(0.229, 0.224, 0.225)">
      Channel standard deviations for normalization (ImageNet defaults).
    </ParamField>

    <ParamField path="interpolation" type="InterpolationMode" default="BILINEAR">
      Interpolation mode for resizing operations.
    </ParamField>

    <ParamField path="hflip_prob" type="float" default="0.5">
      Probability of applying random horizontal flip.
    </ParamField>

    <ParamField path="auto_augment_policy" type="str" optional>
      Auto-augmentation policy to use. Options:
      - `"ra"`: RandAugment
      - `"ta_wide"`: TrivialAugmentWide
      - `"augmix"`: AugMix
      - Other AutoAugment policy names
      - `None`: No auto-augmentation
    </ParamField>

    <ParamField path="ra_magnitude" type="int" default="9">
      Magnitude for RandAugment (used when `auto_augment_policy="ra"`).
    </ParamField>

    <ParamField path="augmix_severity" type="int" default="3">
      Severity level for AugMix (used when `auto_augment_policy="augmix"`).
    </ParamField>

    <ParamField path="random_erase_prob" type="float" default="0.0">
      Probability of applying random erasing augmentation.
    </ParamField>

    <ParamField path="is_bcos" type="bool" default="False">
      Whether to apply B-cos specific transforms. When `True`:
      - Skips normalization (not needed for B-cos)
      - Adds `AddInverse()` transform to double channels
    </ParamField>
  </Expandable>

  **Example:**
  ```python
  from bcos.data.presets import ImageNetClassificationPresetTrain

  # Standard B-cos training transforms
  train_transform = ImageNetClassificationPresetTrain(
      crop_size=224,
      auto_augment_policy="ra",
      random_erase_prob=0.1,
      is_bcos=True
  )

  # Apply to an image
  transformed = train_transform(pil_image)
  ```
</ParamField>

### ImageNetClassificationPresetEval

<ParamField path="ImageNetClassificationPresetEval" type="class">
  Evaluation transform preset for ImageNet classification.

  <Expandable title="Parameters">
    <ParamField path="crop_size" type="int" required>
      Size for center cropping (typically 224 or 176).
    </ParamField>

    <ParamField path="resize_size" type="int" default="256">
      Size to resize the shorter edge to before center cropping.
    </ParamField>

    <ParamField path="mean" type="tuple" default="(0.485, 0.456, 0.406)">
      Channel means for normalization (ImageNet defaults).
    </ParamField>

    <ParamField path="std" type="tuple" default="(0.229, 0.224, 0.225)">
      Channel standard deviations for normalization (ImageNet defaults).
    </ParamField>

    <ParamField path="interpolation" type="InterpolationMode" default="BILINEAR">
      Interpolation mode for resizing operations.
    </ParamField>

    <ParamField path="is_bcos" type="bool" default="False">
      Whether to apply B-cos specific transforms. When `True`, uses `AddInverse()` instead of normalization.
    </ParamField>
  </Expandable>

  <Expandable title="Methods">
    <ParamField path="no_scale" type="method">
      Apply transforms without resize and crop operations.

      **Parameters:**
      - `img`: PIL Image

      **Returns:**
      - Tensor: Transformed image
    </ParamField>

    <ParamField path="transform_with_options" type="method">
      Apply transforms with optional resize and center crop.

      **Parameters:**
      - `img`: PIL Image
      - `center_crop` (bool): Whether to apply center crop. Default: `True`
      - `resize` (bool): Whether to apply resize. Default: `True`

      **Returns:**
      - Tensor: Transformed image
    </ParamField>

    <ParamField path="with_args" type="method">
      Create a new instance with updated arguments.

      **Parameters:**
      - `**kwargs`: Arguments to override

      **Returns:**
      - ImageNetClassificationPresetEval: New instance with updated parameters
    </ParamField>
  </Expandable>

  **Example:**
  ```python
  from bcos.data.presets import ImageNetClassificationPresetEval

  eval_transform = ImageNetClassificationPresetEval(
      crop_size=224,
      resize_size=256,
      is_bcos=True
  )

  # Standard transform
  transformed = eval_transform(pil_image)

  # Transform without resizing
  transformed = eval_transform.transform_with_options(
      pil_image, 
      resize=False, 
      center_crop=True
  )
  ```
</ParamField>

## CIFAR-10 Presets

### CIFAR10ClassificationPresetTrain

<ParamField path="CIFAR10ClassificationPresetTrain" type="class">
  Training transform preset for CIFAR-10 classification.

  Applies standard CIFAR-10 augmentations: random horizontal flip and random crop with padding.

  <Expandable title="Parameters">
    <ParamField path="mean" type="tuple" default="(0.4914, 0.4822, 0.4465)">
      Channel means for normalization (CIFAR-10 defaults).
    </ParamField>

    <ParamField path="std" type="tuple" default="(0.2470, 0.2435, 0.2616)">
      Channel standard deviations for normalization (CIFAR-10 defaults).
    </ParamField>

    <ParamField path="is_bcos" type="bool" default="False">
      Whether to apply B-cos specific transforms. When `True`, uses `AddInverse()` instead of normalization.
    </ParamField>
  </Expandable>

  **Example:**
  ```python
  from bcos.data.presets import CIFAR10ClassificationPresetTrain

  train_transform = CIFAR10ClassificationPresetTrain(is_bcos=True)
  transformed = train_transform(pil_image)
  ```
</ParamField>

### CIFAR10ClassificationPresetTest

<ParamField path="CIFAR10ClassificationPresetTest" type="class">
  Test/evaluation transform preset for CIFAR-10 classification.

  Simple transform pipeline with just tensor conversion and normalization/AddInverse.

  <Expandable title="Parameters">
    <ParamField path="mean" type="tuple" default="(0.4914, 0.4822, 0.4465)">
      Channel means for normalization (CIFAR-10 defaults).
    </ParamField>

    <ParamField path="std" type="tuple" default="(0.2470, 0.2435, 0.2616)">
      Channel standard deviations for normalization (CIFAR-10 defaults).
    </ParamField>

    <ParamField path="is_bcos" type="bool" default="False">
      Whether to apply B-cos specific transforms. When `True`, uses `AddInverse()` instead of normalization.
    </ParamField>
  </Expandable>

  **Example:**
  ```python
  from bcos.data.presets import CIFAR10ClassificationPresetTest

  test_transform = CIFAR10ClassificationPresetTest(is_bcos=True)
  transformed = test_transform(pil_image)
  ```
</ParamField>
