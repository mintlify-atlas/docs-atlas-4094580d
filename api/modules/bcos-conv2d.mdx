## Overview

`BcosConv2d` is a 2D convolutional layer that uses unit norm weights and a cosine similarity activation function. The cosine similarity is calculated between the convolutional patch and the weight vector, and the output is scaled by the cosine similarity raised to the power of (B-1).

See the paper for more details: [B-cos Networks: Alignment is All We Need for Interpretability](https://arxiv.org/abs/2205.10268)

## BcosConv2d

```python
from bcos.modules import BcosConv2d
```

<ParamField path="b" type="int | float" default="2">

  Number of MaxOut units to use. If 1, no MaxOut is used. Must be greater than 0.
</ParamField>

### Usage Example

```python
import torch
from bcos.modules import BcosConv2d

# Create a B-cos convolutional layer
conv = BcosConv2d(
    in_channels=3,
    out_channels=64,
    kernel_size=3,
    stride=1,
    padding=1,
    b=2
)

# Forward pass
input_tensor = torch.randn(8, 3, 224, 224)
output = conv(input_tensor)  # Shape: (8, 64, 224, 224)
```

### Building Custom Models

You can use `BcosConv2d` as a drop-in replacement for `nn.Conv2d` when building custom B-cos models:

```python
import torch.nn as nn
from bcos.modules import BcosConv2d, BcosSequential
from bcos.modules.norms import DetachableGNLayerNorm2d

class CustomBcosBlock(nn.Module):
    def __init__(self, in_channels, out_channels, b=2):
        super().__init__()
        self.conv1 = BcosConv2d(
            in_channels,
            out_channels,
            kernel_size=3,
            padding=1,
            b=b
        )
        self.norm = DetachableGNLayerNorm2d(out_channels)
        self.conv2 = BcosConv2d(
            out_channels,
            out_channels,
            kernel_size=3,
            padding=1,
            b=b
        )
    
    def forward(self, x):
        x = self.conv1(x)
        x = self.norm(x)
        x = self.conv2(x)
        return x

class CustomBcosModel(nn.Module):
    def __init__(self, num_classes=1000):
        super().__init__()
        self.features = BcosSequential(
            BcosConv2d(3, 64, kernel_size=7, stride=2, padding=3, b=2),
            DetachableGNLayerNorm2d(64),
            nn.MaxPool2d(3, stride=2, padding=1),
            CustomBcosBlock(64, 128),
            nn.AdaptiveAvgPool2d(1),
        )
        from bcos.modules import BcosLinear, LogitLayer
        self.classifier = nn.Sequential(
            nn.Flatten(1),
            BcosLinear(128, num_classes, b=2),
            LogitLayer()
        )
    
    def forward(self, x):
        x = self.features(x)
        x = self.classifier(x)
        return x
```

## BcosConv2dWithScale

<Warning>
  `BcosConv2dWithScale` is deprecated and will be removed in a future version. Use `BcosConv2d` instead.
</Warning>

```python
from bcos.modules import BcosConv2dWithScale
```

This variant includes an additional scale parameter for backward compatibility with the original B-cos v1 implementation. It inherits all parameters from `BcosConv2d` and adds:

<ParamField path="scale" type="Optional[float]">
  Custom scale value. If None, uses default initialization: `(kernel_size * sqrt(in_channels)) / scale_factor`
</ParamField>

<ParamField path="scale_factor" type="int | float" default="100.0">
  Denominator for default scale calculation
</ParamField>

## NormedConv2d

A standard 2D convolution with unit norm weights (internal component used by `BcosConv2d`):

```python
from bcos.modules.bcosconv2d import NormedConv2d

# Standard Conv2d but weights are normalized to unit norm
conv = NormedConv2d(in_channels=3, out_channels=64, kernel_size=3)
```
