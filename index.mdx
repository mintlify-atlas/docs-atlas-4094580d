    Quick start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with B-cos Networks in minutes
  </p>

  <Steps>
    <Step title="Install the package">
      Install B-cos Networks using pip. The package includes all pretrained models and core utilities.

      ```bash
      pip install bcos
      ```
    </Step>
    
    <Step title="Load a pretrained model">
      Load any of our 40+ pretrained models directly via torch.hub. No additional configuration needed.

      ```python
      import torch

      # Load a pretrained B-cos ResNet-50
      model = torch.hub.load('B-cos/B-cos-v2', 'resnet50', pretrained=True)
      ```
    </Step>
    
    <Step title="Generate explanations">
      Use the built-in `.explain()` method to visualize what your model is focusing on.

      ```python
      from PIL import Image
      import matplotlib.pyplot as plt

      # Load and transform image
      img = model.transform(Image.open('image.jpg'))
      img = img[None].requires_grad_()

      # Generate prediction and explanation
      model.eval()
      expl_out = model.explain(img)
      
      print(f"Prediction: {expl_out['prediction']}")
      plt.imshow(expl_out['explanation'])
      plt.show()
      ```
      
      <Note>
        The explanation heatmap shows which regions of the input image contributed to the model's prediction, providing direct insight into the decision-making process.
      </Note>
    </Step>
  </Steps>
